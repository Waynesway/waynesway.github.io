折腾 Windows 系统：一次失败的回退尝试

Tuesday, Feb 10, 2026

我的电脑目前运行的是 Windows 10 LTSC 2019，它流畅且稳定，没出过任何问题，我很满意。

但今天，不知道为什么，我产生了一种强烈的欲望，想把系统换回 Windows 7。那个系统我用了 12 年多，它既快又稳，我很怀念旧时光的那种感觉。

决定一下，我就开始动手了。

我懒得用 U 盘启动到 PE 系统，就在网上找了一个“一键安装器”和一个 Windows 7 的 .esd 文件。我点了一下，噩梦就开始了。

软件运行完重启后，我进入了 Win7，但发现我的一个 HDD 硬盘无法被识别。搜索后发现可能是分区格式的问题，除了格式化硬盘似乎没有别的解决办法。

纠结了一阵后，我不得不重新使用 WEPE 把系统装回了 Windows 10。

人老了，可能就没有欲望去学习新东西，且害怕改变。今天我尝试做出改变，但结果似乎证明：最好还是什么都别改。

Win10 安装完之后不能访问共享的解决办法

Tuesday, Feb 10, 2026

这篇文章记录了 Windows 10 安装完成后无法访问网络共享的解决办法。

1. 打开 SMB 协议

在控制面板中启用 SMB 1.0/CIFS 文件共享支持。

路径：控制面板\程序\程序和功能 -> 启用或关闭 Windows 功能 -> 勾选 SMB 1.0/CIFS 文件共享支持。

2. 组策略中打开“启用不安全来宾登录”

如果对方共享没有设置密码，Win10 默认会拦截不安全的来宾登录。

操作步骤：

  1. 按下 Win + R，输入 gpedit.msc 打开组策略编辑器。
  2. 依次选择：计算机配置 -> 管理模板 -> 网络 -> Lanman 工作站。
  3. 在右侧找到 “启用不安全的来宾登录”，双击打开。
  4. 选择 “已启用”，点击确定。
