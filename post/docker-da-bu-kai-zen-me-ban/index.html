Docker打不开怎么办

2025-09-24 1 min read

我本机一般安装那个shellcrah 为了解决恶心的docker被墙打不开，我找过好多网络的第三方的加速的网站，时灵时不灵，索性直接把我本地的环境关上代理了，这样妈妈再也不用担心我pull不下来镜像了。
ai了一下设置步骤如下

Docker 代理：
sudo mkdir -p /etc/systemd/system/docker.service.d
sudo nano /etc/systemd/system/docker.service.d/http-proxy.conf
添加：
[Service]
Environment="HTTP_PROXY=http://192.168.3.4:7890"
Environment="HTTPS_PROXY=http://192.168.3.4:7890"
Environment="NO_PROXY=localhost,127.0.0.1,.qq.com,.tencent.com,.tuna.tsinghua.edu.cn,.docker.io,*.docker.com"
重启：
bashsudo systemctl daemon-reload
sudo systemctl restart docker
